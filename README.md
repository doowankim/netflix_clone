# netflix_clone 제작

## Tool
* Language & Framework : React.js
* Web Skills : HTML, CSS
* Database api : [https://www.themoviedb.org](https://www.themoviedb.org) 

## 기획
### 1.의도
* netflix 같은 영화, TV 프로그램을 공유 DB API를 이용하여 개발해보기 위함
* react.js를 해봄으로써 나의 스킬을 쌓기 위함
